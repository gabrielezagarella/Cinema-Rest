# Cinema-Rest
Example API based on the philosophy Rest. 

Language used Javascript, 
Database used mongodb.

NPM used:

- Node,

- Express,

- Mongoose,

- Nodemon,

- Body-parser.

Routes:

- app.use ("/ cinema", cinemaRouter),

- app.use ("/ films", filmsRouter),

- app.use ("/ users", userRouter),

- app.use ("/ tickets", ticketRouter).
